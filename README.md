## ðŸ› ï¸ Setup & Installation

To run this project locally, you'll need a local server environment like **XAMPP**, **WAMP**, or **MAMP**.

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/salahuddinselim/PHP_Learn.git
   ```
2. **Setup Local Server**:
   - Move the project folder to your server's root directory (e.g., `htdocs` for XAMPP).
   - Start the **Apache** and **MySQL** modules.
3. **Database Configuration**:
   - Open `phpMyAdmin` and create a database named `selim` (as referenced in `sql/connect.php`).
   - Check individual script files in the `sql/` folder for specific table requirements.
4. **Access in Browser**:
   - Navigate to `http://localhost/PHP_Learn` in your web browser.
